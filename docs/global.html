AudioReader(input, audioCtx) → {Promise}

Play a AIFF, CAFF, IFF, AU, WAVE audio files from an url or from a buffer.

AudioReader("http://some.audio/file.ext").then(function(result) {

    // play the sample buffer
    var aCtx = result.audioContext,          // get shared audio context
        source = aCtx.createBufferSource();  // create a playback buffer

    source.buffer = result.buffer;           // set our sample data
    source.connect(aCtx.destination);        // default output
    source.start(0);                         // play - voila!
});

If AudioContext is not found in the browser an error is thrown (un-prefixing is applied internally).

Parameters:
Name Type Argument Description
input string | ArrayBuffer

url or an existing ArrayBuffer with file data loaded into it
audioCtx AudioContext <optional>

optionally provide an AudioContext. If none is provided one will be created

render(options) → {HTMLCanvasElement}

Pass in object with the property data set with the result object returned from audio-reader.

Increasing quality will give a more accurate result, but slow down render time. Likewise, to speed up render you can set maxChannels to f.ex. 1.
